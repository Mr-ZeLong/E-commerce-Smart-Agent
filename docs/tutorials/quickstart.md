# 快速开始

## 环境要求

- Python 3.12+
- Node.js 20+
- PostgreSQL 16
- Redis 7+
- Qdrant 1.16+

## 一键启动（推荐首次使用）

```bash
./start.sh
```

启动后访问：
- API: http://localhost:8000
- API 文档: http://localhost:8000/docs
- C端用户界面: http://localhost:8000/app
- B端管理后台: http://localhost:8000/admin
